================================================================================
JIYASYS PROJECT OVERVIEW & ARCHITECTURE MAP
================================================================================
Last Updated: 2026-05-11
Project Status: Initial Architecture & Homepage Setup
Tech Stack: Next.js 15, TypeScript, Tailwind CSS 4, GSAP, Framer Motion, Lenis.

--------------------------------------------------------------------------------
CORE DIRECTORY STRUCTURE
--------------------------------------------------------------------------------
/src/app         -> Main application routes and global layouts.
/src/components  -> Reusable UI components and page sections.
/src/hooks       -> Custom React hooks for animations and logic.
/src/lib         -> Utility functions and external configurations.
/src/types       -> TypeScript interface and type definitions.
/backend         -> Node.js/Express server logic and MongoDB models.

--------------------------------------------------------------------------------
FILE-BY-FILE BREAKDOWN
--------------------------------------------------------------------------------

1. [src/app/layout.tsx]
   - Purpose: Root layout configuration.
   - Features: Implements global fonts (Inter & Outfit), Custom Cursor, and Smooth Scrolling (Lenis).
   - Tools: Next.js Metadata API, Google Fonts.

2. [src/app/globals.css]
   - Purpose: Global styling and theme configuration.
   - Features: Defines the "Futuristic Dark Luxury" theme using Tailwind 4 CSS variables.
   - Design: Deep black background (#030303), Electric Blue/Purple/Cyan accents, Noise textures.

3. [src/components/shared/SmoothScroll.tsx]
   - Purpose: Global smooth scrolling wrapper.
   - Tools: Lenis (React version).
   - Feel: Premium, inertial scrolling effect across the entire site.

4. [src/components/shared/CustomCursor.tsx]
   - Purpose: Interactive UI element.
   - Tools: GSAP.
   - Features: Magnetic-style follower, reacts to links/buttons by scaling up.

5. [src/components/shared/Navbar.tsx]
   - Purpose: Main navigation and branding.
   - Features: Animated Mega Menu with categories (About, Services, Solutions).
   - Tools: Framer Motion (AnimatePresence), Lucide React Icons.

6. [src/components/shared/Footer.tsx]
   - Purpose: Site-wide bottom navigation.
   - Features: 4-column layout, Social icons, Glassmorphism blur effects.
   - Tools: Lucide React.

7. [src/components/sections/Hero.tsx]
   - Purpose: Primary landing section of the Home page.
   - Features: Cinematic text reveal, animated background gradients, primary CTAs.
   - Tools: Framer Motion.

8. [src/components/sections/TrustedBy.tsx]
   - Purpose: Social proof and credibility.
   - Features: Infinite scrolling logo marquee.
   - Tools: Framer Motion (Keyframes).

9. [src/components/sections/Services.tsx]
   - Purpose: Showcasing core capabilities.
   - Features: Interactive glass-card layout, hover-glow effects.
   - Tools: Framer Motion, Tailwind Utility Classes.

10. [src/components/sections/Industries.tsx]
    - Purpose: Domain expertise showcase.
    - Features: Modern Bento Grid layout with hover-scale images.
    - Tools: Framer Motion, CSS Grid.

11. [backend/server.js]
    - Purpose: Backend entry point.
    - Tools: Express.js, Mongoose (MongoDB).
    - Features: Basic API route and database connection logic.

--------------------------------------------------------------------------------
TECHNOLOGIES & TOOLS USED
--------------------------------------------------------------------------------
- Next.js 15: Latest framework for speed and SEO.
- Tailwind CSS 4: Next-gen styling with CSS-variable based configuration.
- GSAP: High-performance animations for the cursor.
- Framer Motion: Smooth, declarative entrance and hover animations.
- Lenis: High-end smooth scrolling experience.
- Lucide React: Clean, futuristic SVG icons.
- MongoDB: Scalable NoSQL database for enterprise data.
